# Explicación rápida del proyecto

Bienvenido a GXBridge


En este apartado se hará una breve explicación sobre nuestro proyecto.

**Explicación Breve**

Este proyecto que nos fue asignado se trata de un sistema de *facturación*

>Cuando hablamos de un sistema de facturación nos referimos a un sistema que se encarga de transformar la información de un lado (en este caso *Xero*) y llevarlo a otro lado (*GTI*) para que haya una conexión entre ambos lugares.

Nosotros nos encargamos de crear esa conexión para que así los datos sean capaces de llegar un lado a otro de manera segura y eficiente.

>En adición a esto, nosotros logramos hacer que toda la información fuese traducida y convertida en un lenguaje en el que GTI fuera capaz de entender lo que se recibía de Xero.
