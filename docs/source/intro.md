
# Introducción

**Introducción a GXBridge**

>***Pregunta 1***
>
>¿Para qué es el proyecto?

Nuestro proyecto es para el enlace de sistemas de facturas de 2 empresas por medio de un sistema avanzado basado en Python. 

>***Pregunta 2***
>
>¿Para quién es proyecto?

En este caso, RSM (que su sistema se llama Xero) y GTI (que poseen un sistema propio de facturas electrónicas).
